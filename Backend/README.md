## Requirements

- .NET 7 SDK
- Microsoft SQL Server

## Getting Started

Ensure dependencies are installed with:

```bash
dotnet restore
```

Run the development server with:

```bash
dotnet run --project ./Assessment.Api
```

### Stack

- .NET 7.0
- EF 7.0
- Serilog, with Seq integration
- OpenTelemetry, with Jaeger integration
- Swagger
- MediatR
